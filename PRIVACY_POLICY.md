## English Version

### Overview

"mdbook theme" is a browser extension that provides useful tools such as JSON formatting, Base64 encoding/decoding, URL encoding/decoding, timestamp conversion, hash calculation, and regular expression testing. We take your privacy very seriously.

### Data Collection

**This extension does not collect any user data.**

Specifically:
- We do not collect personal identification information
- We do not collect browsing history
- We do not collect any text or data you input
- We do not use cookies or any tracking technologies
- We do not share any data with third parties

### Data Processing

All data processing is done locally in your browser:
- JSON formatting, Base64 conversion, URL encoding/decoding, timestamp conversion, hash calculation, and regex testing are all performed locally
- All data you input exists only in your browser's memory
- All input data is cleared when you close the extension popup
- No data is sent to external servers

### Permissions

This extension only requests minimal necessary permissions:
- **No special permissions required**: This extension does not need access to your browsing data, tabs, or any website content

This extension does not use any third-party services, analytics tools, or advertising networks.
